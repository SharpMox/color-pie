# Archetype columns — proposed picks (Cheapest / Iconic)

Per cell: On-Rate is the existing (already-optimized) card. Cheapest = strictly-mono lowest CMC
that does the effect as its point, **drawbacks allowed** (ties keep On-Rate). Iconic = the famous
mono staple (editorial — flagged ⚠ where it's a judgment call).

`= On-Rate` means the archetype resolves to the same card (allowed; tiny pool / on-rate is also cheapest/famous).

### What this batch demonstrates
- **Cheapest can beat On-Rate** (Masterwork 1<3, Pact 0<2) — the "cheap because of a downside" picks.
- **Iconic frequently = On-Rate** — the famous card was often already chosen as the purest On-Rate.
  Where it's a real judgment call it's flagged ⚠ for your review.
- Small pools (colorless, tight keyword effects) converge — that's expected and fine (dups allowed).
