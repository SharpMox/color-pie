## Model (v2 — incorporates your 3 decisions)
- **Template = 5 variants** (Has-temporary DROPPED): `Grant`×{perm,temp} · `Has` perm · `Remove`×{perm,temp}.
- **Defender Remove** = "can attack as though it didn't have defender" → counted as Remove (● real).
- **Protection** splits by a 9-member "from" set (White/Blue/Black/Red/Green/Everything/Multicolored/Monocolored/Artifacts/Colorless... 9 of the 10 non-trivial), each ×5 variants.
- Rows: original keyword row deleted; flat rows tagged `Base Keyword`; title `Keyword — Action, duration`. Optional `Interaction`/`Duration` selects. Protection rows also tag `Protection From`.
- **Inherit (⭐):** original row's example cards/rankings/archetype picks → `Has, permanent`. **Protection has no single inheritor under the from-split — see note below.**
- Flags: ● common · ◐ sparse · ○ ~unprinted (kept, flagged).
