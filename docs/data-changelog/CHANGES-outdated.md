# CHANGES — Outdated archetype fill

The **Outdated** archetype = the recognizable, older / below-rate card for an effect (the inverse of On-Rate — e.g. Shivan Dragon as the dated fatty, Day of Judgment as the worse Wrath). Pool-authoritative: every pick was confirmed present in that effect’s strict Scryfall pool, correct color, on-pie (WUBRG/◇ where the color is Primary/Secondary/Tertiary, + 🟡 where an On-Rate exists), and never equal to the On-Rate. Two-tier pass: **24 hand-curated famous picks** (with rationale) + a **broader fill** (most recognizable older below-rate card per remaining cell; cells whose only candidates were stronger than On-Rate or unrecognizable jank were left blank).

**574 picks applied** across 233 effects. ~147 in-scope cells left blank (no suitable older card).
