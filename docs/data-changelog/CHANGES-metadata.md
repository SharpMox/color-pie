## Category
- **Keyword Grant/Has** → inherit base keyword's bucket (Flying→Evasion, etc.; resolved at apply by reading each primary).
- **Keyword Remove** → Flying/Hexproof/Indestructible → **Disruption**; Defender → **Combat Abilities**.
- **Protection (50) + choose-a-color (5)** → **Protection & Defense** (original deleted, no inherit).
- **Stage-B** → inherit base effect's bucket (Counterspell→Disruption, etc.).
