# Color Pie Explorer

An interactive reference for Magic: The Gathering's **mechanical color pie** — every effect
mapped to the colors that produce it, ranked **Primary / Secondary / Tertiary**, each with a
representative example card.

**▶ Live site: https://sharpmox.github.io/color-pie/**

Based on Mark Rosewater's *Mechanical Color Pie 2021*. Static HTML + vanilla JS; the data in
`data/colorpie.js` is generated from Notion CSV exports by `scripts/build-data.js`.
